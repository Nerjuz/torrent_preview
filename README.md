# Torrent Posters Preview

**Torrent Posters Preview** is a lightweight Chrome extension that enhances your browsing experience on Lithuanian torrent trackers by automatically fetching and displaying poster previews directly in the torrent list.

No more clicking back and forth to see what a movie or game looks like!

## Features

-   **Automatic Previews:** Posters are inserted directly into the torrent list rows.
-   **Smart Caching:** Images are cached locally to minimize network requests and load lists instantly.
-   **Supported Sites:**
    -   `torrent.lt`
    -   `linkomanija.net`
-   **Toggleable:** Easily enable or disable the extension via the popup menu.
-   **Privacy Focused:** No tracking, no external servers, all data stays on your device.

## Installation

### From Source (Developer Mode)

1.  Clone or download this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top-right corner.
4.  Click **Load unpacked**.
5.  Select the folder containing this extension's files.

## Usage

1.  Visit any supported torrent site (e.g., `torrent.lt`).
2.  Browse the torrent lists as usual.
3.  Posters will automatically appear next to the torrent names.
4.  To clear the cache or disable the extension, click the extension icon in your browser toolbar.

## Privacy

This extension respects your privacy.
-   **No Analytics:** We do not track your browsing history.
-   **Local Storage:** All cached images and settings are stored locally on your machine.
-   **Direct Connections:** Image requests are made directly from your browser to the torrent sites.
